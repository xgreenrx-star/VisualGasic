# Visual Gasic Documentation

Welcome to the official documentation of **Visual Gasic**, the free and open source community-driven BASIC extension for the Godot Engine.

**Visual Gasic** allows you to create games and applications using a syntax inspired by classic BASIC (VB6, QBasic), while leveraging the full power of Godot's modern rendering, physics, and export capabilities.
