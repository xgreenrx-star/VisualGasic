# World Player Module - Reference Code

This directory contains archived legacy code from the original implementation.
These files are preserved for reference and experimentation with the original 9-mode system.

## Files

- `player_interaction_legacy.gd` - Original 2300+ line player interaction script with all 9 modes
- `character_body_legacy.gd` - Original character movement script

## Note

These files are NOT active in the project. They are reference only.
